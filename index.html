Ontology for resolving 'Bag Of Files (bof)'

Creator
indigo-lab
Created

2024-10-30

Description

DCAT で言及されている Bag Of Files は データカタログやレポジトリにおいて単にファイルが集められているものを指し、 個々のファイル間の関係は整備されておらず、dcat:Dataset と dcat:Distribution が正しく区別されていません。

このオントロジーは Bag of Files を bof:Bag や bof:File といったクラスで記録し、 それらを Findable かつ Reusable な dcat:Dataset や dcat:DatasetSeries に再編成する過程で使用することを目的としています。

Classes

bof:Bag c

IRI https://indigo-lab.github.io/bof#Bag
Description

コンテナ、フォルダ、アーカイブ。他のファイルやコンテナ、フォルダ、アーカイブを保持する。dcat:Dataset のサブクラス。
Example 
# Plateau プロジェクト全体を指すコンテナ

<urn:uuid:01923271-71e1-7223-a5c3-2f2bd0b7fb3a>
  a bof:Bag ;
  dcat:landingPage
    <https://www.mlit.go.jp/plateau/> ,
    <https://www.geospatial.jp/ckan/dataset/plateau> ;
.  

# Plateau 東京23区2020年 一式を収録した zip

<ni:///sha-256;gS2jAHPpmJ_f5SYAEFhhtc1qXQB2ZrZjsJiLMmpPmWw>
  a bof:Bag ; 
  rdfs:label "data/zip/13100_tokyo23-ku_2020_citygml_3_2_op.zip" ;
  dcat:landingPage
    <https://www.geospatial.jp/ckan/dataset/plateau-tokyo23ku/resource/0bab2b7f-6962-41c8-872f-66ad9b40dcb1> ;
  dcat:distribution [
    a dcat:Distribution ;
    dcat:downloadURL
      <https://gic-plateau.s3.ap-northeast-1.amazonaws.com/2020/13100_tokyo23-ku_2020_citygml_3_2_op.zip> ;
  ] ;
  dct:spatial <http://data.e-stat.go.jp/lod/sac/C13000> ;
  dct:temporal <http://dbpedia.org/resource/2020> ;
  dct:isPartOf <uuid:01923271-71e1-7223-a5c3-2f2bd0b7fb3a> ;
Note

bof:Bag の ID は UUID で生成するものとします。

ただし、zip, 7z, tar, tar.gz といったアーカイブファイルを bof:Bag としてモデリングする場合には 当該ファイルの SHA256 を算出して 'Named Information' Identifier を ID として使用します。
Sub Class Of dcat:Dataset

bof:File c

IRI https://indigo-lab.github.io/bof#File
Description

ファイルの実体。dcat:Dataset のサブクラス。
Example 
# zip 内ファイル

<ni:///sha-256;n6dRsLyGG5guU-l8jC1JZGGO3uEVgQq85LNnpZhtQX0>
    a bof:File ;
    rdfs:label "data/zip/13100_tokyo23-ku_2020_citygml_3_2_op.zip#13100_tokyo23-ku_2020_citygml_3_2_op/udx/bldg/53392546_bldg_6697_2_op.gml",
    dct:title "data/gml/13100_tokyo23-ku_2020_citygml_3_2_op/udx/bldg/53392546_bldg_6697_2_op.gml" ;
    dcat:landingPage
        <https://www.geospatial.jp/ckan/dataset/plateau-tokyo23ku/resource/0bab2b7f-6962-41c8-872f-66ad9b40dcb1> ;
    dcat:distribution [
        a dcat:Distribution ;
        dcat:downloadURL
            <https://gic-plateau.s3.ap-northeast-1.amazonaws.com/2020/13100_tokyo23-ku_2020_citygml_3_2_op.zip> ;
    ] ;
    dct:isPartOf <ni:///sha-256;gS2jAHPpmJ_f5SYAEFhhtc1qXQB2ZrZjsJiLMmpPmWw> ;
    dct:spatial <http://data.e-stat.go.jp/lod/gridCode/G2053392546> ;
    dct:temporal <http://dbpedia.org/resource/2020> ;
.
Note

bof:File の ID は当該ファイルの SHA256 を 'Named Information' Identifier として URI 化したものを使用します。

zip, 7z, tar, tar.gz といったアーカイブファイルには通常 bof:Bag を使用し、 アーカイブに含まれるファイルの実体を bof:File としてモデリングします。 アーカイブ内のファイル階層を bof:Bag を使用して表現することも可能です。

ただし、GTFS や Office Open XML のように 仕様上 zip 等でパッケージされているものの、 zip そのものを dcat:Dataset とみなすことが妥当なファイルの場合には bof:File を使用します。
Sub Class Of dcat:Dataset

bof:Slice c

IRI https://indigo-lab.github.io/bof#Slice
Description

bof:File の集合。dcat:Dataset のサブクラス。
Example 
# SPARQL によって生成される bof:Slice

<urn:uuid:01923271-9d83-7ffd-95d9-eb1c88f6a9d5>
    a bof:Slice ;
    rdfs:label "13100_tokyo23-ku_2020_citygml_3_2_op.zip のうち bldg のみ" ;
    dct:hasPart <ni:...> , <ni:...> ... ;
    prov:wasGeneratedBy [
        a schema:SearchAction ;
        schema:query "SELECT ?file WHERE {?file a bof:File ; dcat:theme citygml:Building}" ;
        schema:target 
    ]
.
Note

bof:Slice はなんらかの条件に従って作成された bof:File の集合です。

bof:Bag の ID は UUID で生成するものとします。
Sub Class Of dcat:Dataset

Object Properties

dct:hasPart op

IRI http://purl.org/dc/terms/hasPart
Note

dct:hasPart は bof:Bag が配下の bof:Bag または bof:File を参照するために使用します。 これによってツリー構造を表現できます。

また、bof:Slice が配下の bof:File を参照する際にも使用されます。

See Also : DCAT3 6.4.21 Property: has part

See Also : DCAT3 7. Use of inverse properties
